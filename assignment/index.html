4월 13일, 함께했던 4개 세션의 핵심 내용을 돌아보세요.
이제 여러분의 업무에 직접 적용해 볼 시간입니다.

Session 01 · AI 우수사례
우유안부 결산서 자동화
영업지원팀 박지혜님
  • 월 85개 결산서 수기작성 → ChatGPT+VBA로 자동 생성
  • 3단계 접근: 요구사항 분석 → 코드 생성 → 검증·적용
  • 정량 성과: 지역사업부 월 결산 수기작업 시간 4시간 단축
💡 업무 본질 파악이 AX 성과를 극대화한다
Session 02 · VBA 첫걸음
엑셀 VBA & 바이브코딩 개념
피플팀 김창중님
  • VBA = 엑셀 내장 자동화 언어, 별도 설치 불필요
  • 바이브코딩 = 자연어 명령으로 AI가 코드 대신 작성
  • Alt+F11(편집) / Alt+F8(실행) 핵심 단축키
💡 AI로 VBA 생성 → 매크로 실행 → 지금 바로 업무에 적용해보세요
Session 03 · VBA 실습
단계별 자동화 실습
피플팀 김창중님
  • 프롬프트 3원칙: 역할 부여 + 데이터 설명 + 결과 명시
  • 실습1~5: 데이터 정리→요약→서식→차트→이메일 발송
  • 저장 시 반드시 .xlsm 형식 (매크로 포함)
💡 코드 이해보다 AI에게 정확히 요청하는 법, 기억하시나요?
Session 04 · AI툴 3선
데이터 분석·시각화·보고서
피플팀 김창중님
  • Claude in Excel: 엑셀 안에서 AI 작업 요청 (추가기능)
  • Gemini Canvas: 파일 업로드 → 대시보드 자동 생성
  • NotebookLM: PDF 업로드 → 요약/Q&A/오디오 변환
💡 사후과제 활용 도구: Gemini Canvas 또는 NotebookLM으로 직접 만들어보세요

Gemini Canvas
Gemini로 3단계면
대시보드 완성!
엑셀 초보도 1분 만에 인터랙티브 대시보드를. CSV/엑셀 파일만 있으면 Gemini Pro가 분석·시각화합니다.
  • 1단계: Pro 모델 + Canvas 기능 선택
  • 2단계: CSV/엑셀 파일 첨부 + 프롬프트 작성
  • 3단계: 차트 수정 후 공유 링크 생성
📎 사업보고서 데이터를 CSV로 정리 → Gemini Canvas 업로드 → 대시보드 자동 생성

심화 활용
